# Manual V6.3 — Fotografias Inteligentes, GPS e IA

## Objetivo
A versão V6.3 da RJP Obras Pro IA acrescenta uma camada de registo fotográfico técnico, permitindo associar fotografias diretamente a obras, medições, autos, diário de obra, não conformidades e relatórios.

## Fluxo recomendado

1. Abrir a obra.
2. Entrar no separador **Fotografias**.
3. Tirar ou importar uma fotografia.
4. Registar GPS.
5. Escolher categoria.
6. Escrever observação técnica.
7. Associar a:
   - medição;
   - artigo do orçamento;
   - auto;
   - diário de obra;
   - não conformidade.
8. Gerar relatório fotográfico.

## Categorias recomendadas

- Estrutura
- Betão
- Armaduras
- Cofragem
- Terraplenagem
- Pavimentos
- Coberturas
- Acabamentos
- Segurança
- Patologias
- Vedações
- AMV
- EDF
- EBTCC
- Inspeções

## Dados por fotografia

Cada registo fotográfico deve guardar:

- ID da obra
- Nome da obra
- Data
- Hora
- Latitude
- Longitude
- Categoria
- Descrição
- Observação técnica
- Estado
- Ligação ao relatório
- Ligação ao Drive

## Integração com IA

A IA poderá ser usada para:

- gerar legenda técnica;
- criar texto para diário de obra;
- identificar possíveis anomalias;
- classificar a fotografia;
- sugerir ações corretivas;
- gerar relatório fotográfico.

## Relatórios

A V6.3 deve preparar relatórios em PDF com:

- capa da obra;
- lista de fotografias;
- data/hora;
- coordenadas GPS;
- legenda;
- observações;
- estado;
- assinatura/responsável.
